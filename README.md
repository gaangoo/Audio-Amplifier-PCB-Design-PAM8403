# Audio-Amplifier-PCB-Design-PAM8403
This is the PCB Design of Audio Amplifier PAM8403 3W two speakers

The design is based on the 3watt double speaker PAM8403 controller based on Altium Designer

A similar design can be found on here at Amazon: [https://www.amazon.com/s?k=pam8403+amplifier+board&crid=3INL9XHOH7OMG&sprefix=pam8403+amplifier+board%2Caps%2C473&ref=nb_sb_noss_1](https://www.amazon.com/s?k=8403+amplifier+board&crid=12J5889DL8ZXR&sprefix=pam8403+amplifier+board%2Caps%2C519&ref=nb_sb_ss_saint-en-refocus-candidate_4_23)

The repository has the following files:
1. Audio Amplifier 3W PAM8403 PCB.PcbDoc: Alitum Design for the PCB
2. Audio Amplifier 3W PAM8403 PCB.STP: The 3D Model for the PCB Design
3. Audio Amplifier 3W PAM8403 Schematic Layout.SchDoc: The Altium Schematic Layout for the design
4. Audio Amplifier 3W PAM8403 Schematic layout. DWG: The drawing for the layout in AutoCAD file format 
5. Audio Amplifier 3W PAM8403.PDF: The PDF format for the schematic design 
6. Audio Amplifier 3W PAM8403.XLSX: The Bill of Materials used in the design
